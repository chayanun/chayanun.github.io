Introduction To JavaScript

JavaScript is a high-level programming language that all modern web browsers support. It is also one of the core technologies of the web, along with HTML and CSS that you may have learned previously. This section will cover basic JavaScript programming concepts, which range from variables and arithmetic to objects and loops.

What you'll learn

  • 1. Introduction and Variables

    In this course, you learn about JavaScript data types, built-in methods, and variables.

  • 2. Conditionals

    Learn how to use if, else if, else, switch, and ternary syntax to control the flow of a program in JavaScript.

  • 3. Functions

    Learn about JavaScript function syntax, passing data to functions, the return keyword, ES6 arrow functions, and concise body syntax.

  • 4. Arrays and Loops

    Learn about arrays, a data structure in JavaScript used to store lists of data. And you will learn how to use for and while loops to execute blocks of code multiple times.

  • 5. Objects and Classes

    Learn about JavaScript ES6 object syntax to model real-world items and how to create classes and use inheritance to minimize redundancy in your code.
